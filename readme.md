# JavaScript Mini Projects

A collection of simple projects for learning JavaScript fundamentals.

## Projects

### 1. Counter

Basic counter with increment, decrement, and reset buttons.

**Learned:**

- Working with variables
- Events (addEventListener)
- DOM manipulation
- Conditionals (if/else)
- Changing styles via JavaScript

**[View Project](./01-counter)**

---

### 2. Tip Calculator

Calculator for computing tips and total bill amount.

**Learned:**

- Working with input elements
- Type conversion (parseFloat)
- Mathematical operations
- Working with CSS classes (classList)
- Data validation

**[View Project](./02-tip-calculator)**

---

### 3. Todo List

Task management app with add, complete, and delete functionality.

**Learned:**

- LocalStorage API
- JSON methods (stringify, parse)
- Array manipulation (push, filter)
- Data persistence
- Dynamic list rendering

**[View Project](./03-todo-list)**

---

### 4. Modal Window

Interactive modal window with open/close functionality.

**Learned:**

- Event handling (click events)
- DOM manipulation (classList methods)
- Working with overlays
- ESC key handling (keyboard events)
- Preventing event bubbling

**[View Project](./04-modal-window)**

---

### 5. Color Generator

Random color palette generator with history tracking.

**Learned:**

- Working with arrays
- Random number generation
- Template literals
- Dynamic content generation
- Clipboard API

**[View Project](./05-color-generator)**

---

### 6. Weather App

Real-time weather application with search and history features.

**Learned:**

- API integration (fetch)
- Async/await
- Error handling (try/catch)
- LocalStorage for search history
- Working with external APIs
- Dynamic UI updates
- Theme switching

**[View Project](./06-weather-app)**

---

### 7. Learning Quiz

Interactive trivia quiz with multiple categories, timer, and score tracking.

**Learned:**

- Single Page Application (SPA) architecture
- Advanced state management
- Timer implementation (setInterval/clearInterval)
- Complex API integration
- Array methods (.map, .filter, .slice, .reverse)
- HTML entity decoding
- LocalStorage for history persistence
- Dynamic screen switching
- Closures and scope management

**[View Project](./07-quiz-app)**

---

## How to Use

1. Clone the repository
2. Open any project folder
3. Launch `index.html` in your browser

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
